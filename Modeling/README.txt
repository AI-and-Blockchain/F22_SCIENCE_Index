# Overview

This directory contains all of the modeling iterations used for the development and calculation of the ScienceIndex. The latest version is Modelv2.Rmd, which includes the data retrieval, weights, and graphs for displaying information regarding the ScienceIndex.

# Installation and Running

To run the following code, download the Modelv2.Rmd file and save it do an accessible directory. Download and install [R-Lang](https://cran.r-project.org/web/packages/rlang/index.html) and [RStudio](https://support--rstudio-com.netlify.app/products/rstudio/download/#download). Open the Modelv2.Rmd file in RStudio and execute the code. Please note the data file "authorsFinal.csv" is not listed in this repository due to its file size. It can be transfered on request of the user.

# System Requirements

- (Recommended) 4gb RAM
- (Required) 10gb storage
